# OpenGL Primitive chooser

## Launch

Open .pro file in the Qt-Creator as project and run

## Functionality

Choose primitives to render:
* Cube
* Tetrahedron
* Circle
* Cylinder

Chose projection:
* Frustum
* Ortho
